Guía General

Cómo está pensado el sistema

1. Actividad

La actividad del staff se mide y sirve como base para mantener o recuperar puntos.

2. Inspección

Los inspectores revisan desempeño, ajustan puntos y avisan cuando haga falta intervenir.

3. Degradación

Al llegar a 0 puntos, el sistema bajará de rango y abrirá una apelación privada.

4. Apelación

El miembro afectado podrá apelar durante 24 horas antes de que se confirme o revierta el caso.

Reglas base de puntos

  • Todos los miembros del staff empiezan con 7 puntos.
  • Los puntos los gestionan los inspectores y admins desde el panel.
  • El máximo de puntos es 7.
  • Al llegar a 0 se abre apelación automática.

Rangos

📌 Rangos reales del servidor

👑
Owner
Máximo nivel de autoridad. Se encargan de dirigir la comunidad, tomar decisiones importantes, supervisar la moderación y coordinar el proyecto junto a Jow.
🛡️
Admin
Segundo nivel de autoridad. Supervisan la moderación, aplican sanciones avanzadas y ayudan a mantener el orden general de la comunidad.
⚙️
Tester
Primer nivel de moderación. Se encargan de tareas básicas como vigilar el chat, reportar problemas y aplicar sanciones simples cuando sea necesario.

Cadena de degradación

Owner → Admin → Tester → Usuario

Cuando un miembro del staff llega a 0 puntos, se abre un canal privado de apelación por 24 horas.

Sistema de apelaciones

Esta sección estará activa una vez que el bot de Discord esté funcionando. Las apelaciones se abrirán automáticamente cuando un miembro llegue a 0 puntos.

¿Cómo funcionará?

  • Al llegar a 0 puntos, se abre automáticamente un canal privado de apelación.
  • El miembro tiene 24 horas para presentar su caso.
  • Un admin revisa y decide si revertir o confirmar la degradación.
  • El resultado se registrará aquí como novedad.
