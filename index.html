Guía General

Cómo está pensado el sistema

1. Actividad

La actividad del staff se medirá en Discord y servirá como base para mantener o recuperar puntos.

2. Inspección

Los inspectores revisarán desempeño, ajustarán puntos y avisarán a los Owners cuando haga falta intervenir.

3. Degradación

Al llegar a 0 puntos, el sistema bajará de rango y abrirá una apelación privada para revisión.

4. Apelación

El miembro afectado podrá apelar durante 24 horas antes de que se confirme o revierta el caso.

Reglas base de puntos

  • Todos los miembros del staff empiezan con 7 puntos.
  • Los puntos bajan automáticamente cada 24 horas si no hay actividad suficiente.
  • Hablar y cumplir funciones podrá sumar una cantidad pequeña de puntos.
  • Los inspectores pueden sumar o restar puntos manualmente con motivo.
  • El máximo de puntos visibles en esta plantilla es 7.
Esta página es la base visual del sistema. Más adelante se conectará al bot y a las cuentas para automatizar todo el flujo.

Flujo operativo recomendado

  1. El bot detecta actividad útil o inactividad del staff.
  2. Los inspectores revisan puntos, contexto y desempeño real.
  3. Si alguien cae a 0, se genera alerta y se abre apelación.
  4. Los Owners revisan el caso y deciden si confirmar o restaurar.
  5. Las renuncias se podrán enviar desde la página para que el bot quite el rol.

Criterios de evaluación

  • Actividad básica en Discord puede sumar una cantidad pequeña como referencia.
  • El cumplimiento real del cargo pesa más que solo escribir mensajes.
  • Los inspectores ajustan puntos cuando detectan buen desempeño o incumplimiento.
  • Todo cambio importante debería registrar autor, motivo y fecha cuando se conecte el sistema real.

Protocolo de apelación

  1. Al llegar a 0 puntos se abre un canal privado de apelación.
  2. Entran el miembro afectado, los inspectores y los Owners.
  3. El caso debe resolverse dentro de un plazo de 24 horas.
  4. Solo los Owners deciden si se confirma la degradación o se restaura el rango.

Renuncia voluntaria

La página tendrá una opción para que cualquier miembro del staff pueda renunciar a su cargo o rango. Cuando el sistema esté conectado, el bot quitará el rol en Discord y dejará registro del movimiento.

Esta sección queda preparada como plantilla. Más adelante se añadirá el botón real de renuncia y el historial correspondiente.
